Microservices with .Net 6

Following tools should be installed on your system to be able to run these examples:
1. Visual Studio Code (https://code.visualstudio.com/)
2. .Net 6 (https://www.bing.com/search?q=.net+6&cvid=08c40a21064e482ca6d41793c26937b5&aqs=edge..69i57j69i59j0l5j69i59j0j69i11004.3043j0j9&FORM=ANAB01&PC=HCTS)
3. Docker (https://www.docker.com/)
4. RabitMQ (https://www.rabbitmq.com/)

This repository contains following projects -
1. BasicContainer - Demonstration of building a very basic container application. This has been extracted from MSDN walkthrough supplied here https://learn.microsoft.com/en-us/dotnet/core/docker/build-container?tabs=windows with minor modifications. This demonstrates how to build a simple containerized console application using .net 6 and docker container.
